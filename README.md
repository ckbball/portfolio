# caesarladion.com

A deliberately simple, single-page portfolio for Caesar Ladion.

## Stack

- One semantic HTML page
- One Tailwind-generated stylesheet
- No runtime JavaScript, framework, content pipeline, or page router
- GitHub Pages hosting

## Local development

```bash
npm install
npm run dev:css
```

In a second terminal:

```bash
npm run serve
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

The build compiles the stylesheet and verifies that the four on-page sections are present and no retired page links remain.
